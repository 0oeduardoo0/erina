Handler¶

class erina.Handler(msg, response)[source]¶

Base class to handlers.

Handlers will receive the incoming messages and they will do something if they recognize some command.

__init__(msg, response)[source]¶

Parses the input message.

Handler instaces are instanced by Bot class when some message are received. Cronjob also instaces Handler classes but it doesn’t gives msg and response params.

Parameters:
  • msg (Message) – fbchat message object
  • response (Response) – Response object
cron(time)[source]¶

Called by cronjob.

If you register a handler as cronjob, cron will call this method every minute.

Parameters:time (datetime) – datetime object
handle()[source]¶

Called by Bot class.

It must return True if the incoming command is recognized and executed.
